正整数

正整数的补码是其本身的二进制形式。

> [!example] 5 的补码
> 
> `5` 的二进制形式：
> ```
> 0000000000000101
> ```
> 其补码：
> ```
> 0000000000000101
> ```

负整数的补码：

1. 其绝对值的二进制形式；
2. 所有位取反；
3. 加一

> [!example] -5 的补码
> `5` 的二进制形式： 
> ```
> 0000000000000101
> ```
> 所有位取反： 
> ```
> 1111111111111010
> ```
> 加一： 
> ```
> 1111111111111011
> ```
